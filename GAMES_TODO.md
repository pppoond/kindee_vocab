# 🎮 Game Feature Ideas (TODO)

ไอเดียสำหรับปรับปรุงระบบ Minigame ให้สนุกและน่าสนใจยิ่งขึ้นสำหรับ Kindee Vocab

## 1. ระบบการต่อสู้ (Combat & Mechanics)
- [ ] **Combo System:** ตอบถูกติดต่อกันจะทำ Damage แรงขึ้น (เช่น x1.2, x1.5, x2) และมี Visual Effect ที่อลังการขึ้น
- [ ] **Elemental Skills:** คำศัพท์บางประเภท (Noun, Verb) อาจจะมีธาตุที่ชนะทางสัตว์ป่าต่างกัน
- [ ] **Stamina & Healing:** เพิ่มขวดเลือด (Potion) ให้ผู้ใช้กดใช้ได้ระหว่างสู้ หรือสุ่มดรอปเมื่อตอบถูก
- [ ] **Critical Hits:** สุ่มการโจมตีที่แรงเป็นพิเศษเมื่อตอบคำศัพท์ระดับยากได้ถูกต้อง

## 2. ระบบความก้าวหน้า (Progression & Rewards)
- [ ] **Experience & Leveling:** เล่นจบเกมแล้วได้ XP เพื่อเลเวลอัพ ยิ่งเลเวลสูงยิ่งปลดล็อกตัวละครหรืออาวุธใหม่
- [ ] **Streak System:** ระบบนับวันที่เล่นติดต่อกัน เพื่อกระตุ้นให้ผู้ใช้กลับมาเล่นทุกวัน (Daily Quest)
- [ ] **Global Leaderboard:** ตารางจัดอันดับคะแนนสูงสุดรายสัปดาห์ในหมู่ผู้ใช้งานทั้งหมด
- [ ] **Collection/Bestiary:** เมื่อปราบสัตว์ป่าได้ จะมีการเก็บสะสมข้อมูลสัตว์ป่าลงในสมุดบันทึก

## 3. โหมดการเล่นใหม่ (New Game Modes)
- [ ] **Flashcard Swipe:** โหมดฝึกฝนก่อนสู้จริง (Swipe Left = จำไม่ได้, Swipe Right = จำได้)
- [ ] **Time Attack:** ตอบคำศัพท์ให้ได้มากที่สุดภายในเวลาที่กำหนด (เช่น 60 วินาที)
- [ ] **Boss Raid:** สัตว์ป่าระดับบอสที่มีเลือดเยอะมาก และต้องตอบคำศัพท์ยากๆ ถึงจะผ่านได้

## 4. ความสวยงามและเสียง (Aesthetics & Audio)
- [ ] **Battle BGM & SFX:** เพิ่มเพลงประกอบแนวผจญภัยและเสียง Effect เมื่อโจมตี/โดนโจมตี
- [ ] **Custom Avatars:** ให้ผู้ใช้เลือกตัวละคร (Knight, Mage, Archer) ก่อนเริ่มเกม
- [ ] **Ambient Effects:** เพิ่มสภาพอากาศหรือบรรยากาศในฉากหลัง (เช่น ฝนตก, หิมะตก) ตามเวลาจริง
